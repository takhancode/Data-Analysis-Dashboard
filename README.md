Mobile Sales Data Dashboard – Power BI
=====================================

Project Overview
----------------
This project presents a complete analysis of mobile sales data using Microsoft Power BI. 
The dashboard provides interactive KPIs, trend insights, brand analysis, customer ratings, 
and payment method breakdowns to help businesses make data‑driven decisions.

DataSet
----------------
-<a href="https://github.com/takhancode/Data-Analysis-Dashboard/blob/main/Mobile%20Sales%20Data.xlsx">DataSet</a>

Key Features
------------
1. KPI Indicators:
   - Total Quantity Sold
   - Total Transactions
   - Average Sales Value
   - Total Sales Revenue

2. Interactive Visuals:
   - Line chart for daily sales trends
   - Bar chart for customer ratings
   - Column chart for sales by mobile model
   - Sales by day name line chart
   - Pie chart for payment methods
   - Scrollable brand‑wise table
   3. Filters & Slicers:
   - Monthly slicers (January–December)
   - Brand and day filters

4. Modern & Clean UI:
   - Rounded cards
   - Professional icons
   - Light theme dashboard

Business Insights (Summary)
---------------------------

 iPhone 12 shows strong performance.
- Apple and Samsung lead in brand sales.
- Payment methods are evenly distributed.
- Customer ratings show positive trends.
- Sales peak on specific days of the month.


How to Use
----------
1. Download or clone the repository.
2. Open the .pbix file in Power BI Desktop.
3. Load or replace the dataset.
4. Refresh visuals to view updated insights.
